# Test Antigravity プロジェクト

このリポジトリは、複数のウェブアプリケーションとツールを含むモノレポ構成のプロジェクトです。

## 📁 プロジェクト構成

### `frontend/`
フロントエンドアプリケーション

### `kiwami01-mvp/`
Kiwami MVP - ランディングページ生成システム
- Next.js ベースのアプリケーション
- AI を活用した LP 自動生成機能
- World Trade Next (WTN) ブランドアイデンティティを採用

### `lark-guide-lp/`
Lark ガイド用ランディングページ

### `prisma/`
データベーススキーマとマイグレーション
